Usage

Loading EEG/ExG Data

  1. Launch the application.
  2. Go to the File menu and select one or more EEG files (.csv, .bdf) to load.
  3. The selected files will appear under Loaded Files in the left panel.

Applying Filters

Navigate to the Filters menu to apply:

  • High-pass / Low-pass filters
  • Notch filter (e.g., to remove 50/60 Hz line noise)
  • Re-referencing options
  • DC offset correction

Channel Selection

  • In the Channels section, select or deselect specific channels.
  • Use Select All or Clear All for quick selection.
  • Only selected channels will be used for visualisation.

Visualization Options

Click the corresponding button to visualize the selected data. You can export plots as image files from the toolbar above each visualization.

Tip: Use the interactive toolbar to zoom, pan, and explore plots in detail.

  • Plot Time Domain – View raw signal over time.
  • FFT Plot – Visualize frequency content using Fast Fourier Transform.
  • Bandpower Visualization – Show power in defined frequency bands (delta, theta, alpha, beta). The plots show the average for the selected channels.
  • Power Spectrum Density – Analyze power distribution over frequency. The plot shows the average for the selected channels.
  • Spectrogram – Display power as a function of time and frequency. The plot shows the average for the selected channels.

Exporting Data

  • Go to the File menu and choose Export as CSV to save the current signal data.
