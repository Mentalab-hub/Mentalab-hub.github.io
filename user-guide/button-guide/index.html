Button Command Guide

The Mentalab Explore device was designed to provide maxmimum flexibility, with minimal size. This page lists how the user can communicate with the device using the button.

  Action   State   Effect
Single click Device off Device will turn on
Double click Memory full (LED blinking pink) Delete flash memory.
Return to advertising mode.
Double click Device in startup Skip advertising.
Go to offline mode.
Double click Device in offline mode Start advertising mode.
Keep recording to flash.
Single click Device recording (online or offline) Generate marker timestamped at button press.
Press & hold All states Terminate recording. Turn off.
Device will blink red three times.
