Presentation®

Prerequisites

Hardware

  • Mentalab Explore Pro Device: An Explore Pro biosignal acquisition system.
  • Mentalab Hypersync Add-on: This hardware is essential for accurately synchronizing LSL markers from Presentation® with the Explore device’s data stream.
  • Computer: To run Presentation®, LSL, and data recording software for the Explore device.

Software

  • Presentation®
  • LabRecorder or alternative software that lets you record LSL streams

How it Works

  1. Presentation®: Runs experiments and can send event information as a Marker stream via LSL (including information about event type, the event code and uncertainty for the event)
  2. Built-in LSL integration: The built-in LabStreamingLayer integration provides real-time streaming of event data with precise timestamps for Presentation®
  3. Mentalab Hypersync: The Hypersync suite is used to merge ExG data with event markers via LSL. Please follow the instructions delivered to you with you Hypersync hardware to achieve wireless, high-precision synchronization

Integrating with Presentation® via LSL

Consult the HyperSync manual for achieving synchronisation between the Mentalab Explore amplifier and LSL.

To use LSL with Presentation®, you can enable sending event data to LSL in the Settings in the Presentation® GUI. The option is listed under the General section. Use the Properties to set options for the LSL integration like stream name and what event data to send. When you run an experiment, make sure to open the LSL stream manually (by clicking Open) or enable the option to open the stream outlet when an experiment is loaded in the LSL settings!

The event data will be available as an LSL Marker stream to any other script or software that makes use of LSL, like LabRecorder or custom scripts using LSL.

You can view available LSL streams and record the Marker stream via LabRecorder (using the stream name set in the LSL properties as described above) alongside the EXG datastream (named after the Explore device) from the synchronized Explore amplifier to record synchronized markers from an experiment and EXG data.

For more information on LSL usage with Presentation®, consult the extensive online manual for Presentation®:

  • Information on the LSL settings in Presentation®
  • Overview of the Presentation® Help Guide

To ensure accurate synchronisation, you must use Mentalab Hypersync and follow the instructions provided alongside your Hypersync hardware.

Alternative (Legacy) Methods: Wired Triggers

While LSL with Hypersync is preferred as it enables fully wireless and highly accurate event marking, direct wired connections are also possible. Please refer to our Event Triggers page for further information.
