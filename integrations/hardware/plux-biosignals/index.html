PLUX Biosignals

Integrating Mentalab with PLUX Biosignals devices

Mentalab is partnering with PLUX Biosignals to cover our user’s multimodal physiology needs.

PLUX Biosignals sensor portfolio includes sensors for

  • electrodermal activity sensors (EDA)
  • respiration sensor belts
  • force sensors
  • temperature sensors
  • blood oxygen sensors (SpO2)
  • and much more

Any combination of these sensors can be connected to a central PLUX Biosignals recording unit, which transmits data wirelessly, just like the Mentalab Explore Pro.

Guide

In this guide, we walk you through how the measurement units can be combined in practice by using Lab Streaming Layer (LSL).

You will need

  • Explore Pro with any channel count (+ Hypersync add-on)
  • PLUX Biosignals recording unit

1. Mentalab Explore Pro Setup

Set up your Explore Pro system for your experiment.

While the Explore Pro can push data to LSL directly, users seeking highly accurate synchronisation between the Explore Pro and LSL should use the Hypersync add-on. If this is the case, follow the Hypersync instructions delivered to you with your Hypersync system.

Turn on the Explore Pro and connect it to Explore Desktop. In the visualization page, head to the top right corner to start pushing your Mentalab ExG data data stream to LSL.

2. PLUX Biosignals Setup

Set up your PLUX Biosignals device for your experiment.

Turn on your PLUX Biosignals device and connect it to OpenSignals. Head to the settings to activate OpenSignals’ LSL feature.

3. Record using LabRecorder

LabRecorder is free software that is used to record one or several LSL streams into a single file (in .xdf file format). Download and install LabRecorder. Select all relevant streams from Mentalab Explore (ExG, Orientation data, Event Markers) and the PLUX Biosignals stream.

Start Recording.

If you require assistance parsing the .xdf file into another file format, please reach out to us at support@mentalab.com
